About This Project

Welcome to the Emoji Detection System! This application uses machine learning to detect and recognize emojis in real-time from video input. You can run a live detection script using your webcam or upload a video file to analyze emoji usage in that video. Our system utilizes a trained model to identify different emojis and display the detected emoji on the video feed.

The project is designed to showcase real-time video processing and machine learning applications using TensorFlow, Keras, and OpenCV. We hope you find it useful and engaging!
